# To-do list manager (TODO) — Група 3

Консольний Python-додаток для керування списком задач, розроблений у межах практичної роботи №6.

## Функціонал програми
* Перегляд поточного списку задач.
* Додавання нових задач.
* Видалення існуючих задач за їхнім порядковим номером.
* Автоматичне збереження всіх змін у текстовий файл `data/todo_list.txt`.
* Пошук задач за ключовим словом (без урахування регістру).

## Команда та розподіл ролей
*Студент А (Team Lead / Інтегратор): Розробка структури проекту, реалізація консольного інтерфейсу користувача (`src/interface.py`), фінальна збірка та інтеграція коду в `main.py`.

*Студент Б (Розробник / Тестировщик): Реалізація логіки роботи з файлами, функцій читання/запису, додавання, видалення та пошуку за ключовими словами (`src/storage.py`), обробка винятків та помилок введення.

## Інструкція із запуску
1. Склонуйте репозиторій або відкрийте папку з проектом у терміналі.
2. Запустіть головний файл програми за допомогою команди:
   python main.py
